# Poolroute


An Application to buy and sell poolroutes


## Installation:



Clone the Repo:
```ruby
git clone git@github.com:lkkadiri/poolroute.git
```

Create The Database
```ruby
rake db:create
```

Migrate the Database and populate all the tables
```ruby
rake db:migrate
```
Run the application
```ruby
rails s
```
